### 常用全局方法
---
#### 注册事件

参数: (String 事件名,函数 事件);  

事件名请参考手册的 界面事件  

实例 :  

```
注册事件(事件名,函数() {

提示.普通("我被调用了 ~");

}); // 注册了一个事件

注册事件(事件名,空); // 覆盖一个事件

```
---
#### 结束界面

关闭当前界面  

实例 : 
```
结束界面(); 

// 结束后脚本会继续运行

返 空;

// return后脚本不会继续运行

```
---
#### 置滑动返回

参数: (boolean 状态);  

设置界面是否可以滑动返回 可以重复设置  

实例 : 
```
置滑动返回(真); // 界面可以滑动返回了
置滑动返回(假); // 界面不能滑动返回了
```
